# VR Museum Project

> **Note: This project uses Git LFS (Large File Storage) to manage large binary assets such as textures, audio files, and Unity-specific files. You must have Git LFS installed before cloning, otherwise these assets will not download correctly.**

## Git LFS Setup

1. **Install Git LFS**
   - Download from [git-lfs.com](https://git-lfs.com) or via Homebrew on Mac:
     ```
     brew install git-lfs
     ```

2. **Enable Git LFS**
   ```
   git lfs install
   ```

3. **Clone the repo as normal**
   ```
   git clone <repo-url>
   ```
   LFS assets will download automatically alongside the repo.

---

If you already cloned before installing Git LFS, run:
```
git lfs pull
```
to fetch the missing assets.
